NeonOcto 提示词优化平台界面描述
=======================

整体布局: 三栏式像素风格设计
---------------------------

1. 左栏 - 导航区
   ✓ 顶部 Cursor/火山引擎/未来平台图标 (48x48像素)
   ✓ 菜单列表垂直排列，当前选中项高亮显示
   ✓ 每个平台对应不同像素风格图标

2. 中栏 - 操作区
   ✓ 顶部平台/模型选择器 (级联式下拉框)
   ✓ 中间提示词编辑区 (支持Markdown格式)
   ✓ 底部测试用例生成区 (表格式布局)
   ✓ 最底部优化按钮 (点击时像素闪烁)

3. 右栏 - 信息区
   ✓ 顶部实时指标卡片 (像素虚线边框)
   ✓ 底部历史记录时间轴 (版本递进显示)

UI元素列表
---------------------------
- 像素按钮: 带有3D浮雕效果，点击时有下沉动画
- 编辑器: 暗色背景，带有行号和代码高亮
- 表格: 像素化边框，斑马纹样式
- 模态框: 全屏覆盖，带有像素化边框
- 下拉框: 展开时显示像素风菜单
- 时间轴: 垂直排列的历史条目，当前版本高亮

色彩方案
---------------------------
- 主背景: #121212 (深灰黑色)
- 面板背景: #1a1a1a (稍浅灰黑色)
- 边框颜色: #4a4a4a (中灰色)
- 主题色: #4a90e2 (蓝色)，用于高亮和按钮
- 强调色: #ff6b6b (红色)，用于重要操作和高亮
- 文本颜色: #ffffff (白色) / #bbbbbb (浅灰) / #888888 (中灰)

交互效果
---------------------------
1. 平台切换: 点击左侧菜单，更新顶部图标和模型选项
2. 模型选择: 点击下拉框展开选项列表
3. 优化流程: 点击优化按钮→闪烁动画→显示对比模态框
4. 测试用例: 可添加/删除表格行
5. 历史版本: 点击右侧版本条目切换查看

此原型完整实现了设计文档中指定的像素风UI，具有高度的复古游戏风格感，同时保持了操作的直观性。
